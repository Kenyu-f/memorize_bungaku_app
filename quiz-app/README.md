# Quiz Application

This is a simple quiz application that allows users to create and manage quizzes. Users can add new quiz questions along with their answers, and view a list of all quizzes created.

## Features

- Add new quizzes with questions and answers.
- Display a list of all quizzes.
- User-friendly interface for quiz management.

## Getting Started

To get a local copy up and running, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/Kenyu-f/quiz-app.git
   ```

2. Navigate to the project directory:
   ```
   cd quiz-app
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Start the application:
   ```
   npm start
   ```

## Usage

- Use the "Add Quiz" form to input a new quiz question and its answers.
- View the list of quizzes displayed on the main page.
